# 🏦 Loan Default Prediction System

A Machine Learning-based web application that predicts the likelihood of a customer defaulting on a loan based on financial and demographic information. The system analyzes factors such as income, credit score, loan amount, employment status, and repayment history to assess risk and support informed lending decisions.

The project trains and compares multiple classification models, including **Logistic Regression** and **Random Forest Classifier**, to identify the most effective approach for loan default prediction. The best-performing model is selected and integrated into an interactive Streamlit dashboard for real-time predictions.

## 🚀 Features

* Loan default risk prediction
* Comparison of Logistic Regression and Random Forest models
* Automatic best model selection
* Interactive Streamlit web interface
* Individual and batch predictions
* Model performance evaluation and visualization
* Risk-based loan assessment
